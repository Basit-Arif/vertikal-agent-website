Security at Vertikal Agent

Security is fundamental to Vertikal Agent. Our customers rely on us to handle sensitive data and mission-critical operations, so we invest heavily in safeguards across infrastructure, process, and people.

Infrastructure

  • Hosted on SOC2, ISO 27001 compliant cloud providers with regional redundancy.
  • Network isolation, firewalls, and continuous monitoring across all environments.
  • Automated backups with encryption at rest and retention policies.

Data security

  • Encryption in transit (TLS 1.2+) and at rest (AES-256).
  • Secrets management using locked-down vaults with rotation policies.
  • Role-based access controls with SSO/MFA options for customer accounts.

Application security

  • Secure SDLC: code reviews, static analysis, dependency checks.
  • Penetration testing by third parties and continuous vulnerability scanning.
  • Audit logging for agent actions and administrative activities.

Compliance

Vertikal Agent is preparing for SOC 2 Type II certification and follows GDPR and CCPA guidance for handling personal data. We support data processing agreements and customer-controlled retention policies.
